# Refine Hype Video

## Goal

Create a short product video that shows Refine as a full agentic software system: it builds new apps, implements features, fixes software bugs, and verifies changes through a workflow people can trust.

The strongest video should use real product footage first, with AI-generated material supporting the edit rather than replacing the product.

## Recommended workflow

1. Capture real product footage.

   Record short clips of the actual app:

   - Dashboard/status cards.
   - New Gap creation.
   - Command Palette.
   - Chat and Plan flow.
   - Quality and Playwright regression checks.
   - System > Processes.
   - Merge/workflow progress.

2. Write a tight script.

   Aim for 45-75 seconds:

   - Problem: software work gets lost between people, tools, branches, and machines.
   - Reveal: Refine turns gaps into verified changes.
   - Proof: show the UI doing real work.
   - Close: Refine is a full agentic software system for building, fixing, and verifying software.

3. Edit with AI assistance.

   Use a real video editor for the final composition. Good options:

   - Adobe Firefly / Premiere for editing, cleanup, captions, and final polish.
   - Runway for short generated b-roll and transitions.
   - Google Flow / Veo for generated cinematic support shots.
   - ElevenLabs for voiceover.

4. Generate only supporting b-roll.

   Use generated video for a few short metaphor shots:

   - Code branches converging.
   - Agents coordinating across machines.
   - Command Palette as a control center.
   - Screenshots passing through a quality gate.

   Avoid generic AI magic visuals. The product footage should carry the proof.

5. Add voice, music, and captions.

   Use a restrained voiceover, low electronic/ambient music, and short captions for the main claims. Keep the tone confident and technical, not movie-trailer exaggerated.

## Creative direction

The video should feel like operator-grade software infrastructure.

Use:

- Real UI closeups.
- Terminal/code moments.
- Fast but readable cuts.
- Captions for key claims.
- Generated clips only as transitions or emotional punctuation.

Avoid:

- Cartoon robots.
- Vague AI magic.
- Stock-looking teams in conference rooms.
- Claims that are not shown by the product.

## Possible titles

- Refine: from software gaps to verified changes.
- Refine is a full agentic software system.
- Build, fix, and verify software with Refine.

## Draft 60-second structure

**0-8s:** show problem.

Software work gets scattered across chats, tickets, branches, machines, and manual QA.

**8-18s:** reveal Refine.

Refine gives that work a system: Gaps, workflow, agents, quality, and review.

**18-40s:** show proof.

Create a Gap, open the Command Palette, plan with chat, let agents work, run Quality, inspect regressions, and review the change.

**40-55s:** show system depth.

Processes, instances, target-app controls, merge flow, logs, and repeatable verification.

**55-60s:** close.

Refine turns software gaps into verified changes.
